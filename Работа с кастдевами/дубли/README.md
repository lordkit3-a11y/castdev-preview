# Дубли и архив

Сюда складываем **реальные дубли** (одинаковое назначение или байт-в-байт) и **устаревшие** инструкции, чтобы не путать с актуальными.

## Корень `CustDev/` и папка `Сайт/` — это не «одинаковые копии»

По именам похоже (`index.html`, `kastdev-roman-…`), но содержимое **разное**:

| Файл в корне | Размер (порядок) | Что это |
|--------------|------------------|---------|
| `index.html` | малый (~0,7 КБ) | **Редирект** на `Работа с кастдевами/Сайт/index.html` (короткий URL на GitHub Pages) |
| `Сайт/index.html` | большой (~50 КБ) | **Настоящая главная** с таблицей и задачами |
| `kastdev-*.html` в корне | малый | **Редиректы** на полные страницы в `Сайт/` |
| `Сайт/kastdev-*.html` | большой | **Полные** страницы кастдевов |

Итого: в `дубли` **не переносим** корневые редиректы — иначе сломаются прямые ссылки вида `…/castdev-preview/index.html`.

## README

- Актуально: **`CustDev/README.md`** (корень репозитория).
- В `Сайт/Служебные/` лежит короткий **`README.md`** — только отсылка к корню и к `ПРАВИЛА_кастдевы.md`.
- Старый текст «как в castdev-public» перенесён: `архив_README_сайта_устаревший.md`.

## Что уже в архиве здесь

- `архив_README_сайта_устаревший.md` — устаревшая инструкция.
- `архив_дубль_gitignore_из_Служебные.txt` — копия второго `.gitignore`, он **полностью совпадал** с корневым `CustDev/.gitignore`; из `Служебные/` удалён намеренно.

**Вне репозитория:** папка `castdev-public` в каталоге `Cursor` — отдельный старый git-клон, не смешивать с `CustDev`.
