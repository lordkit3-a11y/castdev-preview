# Что сделать тебе один раз (я уже сделал локальную часть)

Локально готово: папка `castdev-public`, файл `index.html`, первый `git commit`.

## Шаг 1 — создать пустой репозиторий на GitHub

1. [github.com/new](https://github.com/new)
2. Имя — то, что согласуем (латиница), например `castdev-preview`
3. **Public**
4. **Без** README, .gitignore, license (репозиторий пустой)
5. **Create repository**

## Шаг 2 — привязать и отправить код

В терминале на Mac (замени `ЛОГИН` и `ИМЯ_РЕПО`):

```bash
cd "/Users/igor/Cursor/castdev-public"
git remote add origin https://github.com/ЛОГИН/ИМЯ_РЕПО.git
git push -u origin main
```

Если спросит пароль — в GitHub нужен **Personal Access Token** (не пароль от сайта):  
Settings → Developer settings → Personal access tokens.

Проще: установить **GitHub Desktop**, открыть эту папку как репозиторий и нажать **Publish**.

## Шаг 3 — включить Pages

Репозиторий → **Settings** → **Pages** → Source: branch **main**, folder **/ (root)** → Save.

Через минуту ссылка: `https://ЛОГИН.github.io/ИМЯ_РЕПО/`
