# Кастдевы → процессы агентства

Здесь копим материалы по созвонам с клиентами и черновики **процессных** задач (не «починить один проект»).

## Ритм

1. После каждого кастдева — внизу артефакта блок **«Процессные задачи (черновик)»**: максимум **3** задачи в формате шаблона.
2. После **10–15** кастдевов — свести все черновики в одну таблицу, пометить повторяемость и **приоритезировать** (топ 3–5 на квартал).

## Файлы

- `ШАБЛОН_после_кастдева.md` — вставлять в конец каждого нового отчёта.
- Превью по Роману (Дарс) лежит в корне воркспейса: `кастдев_Роман_превью.html` (дублировать сюда при желании).

## Git

При необходимости сделай из этой папки отдельный репозиторий:

```bash
cd кастдев && git init
```
