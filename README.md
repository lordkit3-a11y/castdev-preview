# Каздев

Структура папки:

- `Примеры` — ориентиры и шаблоны. Эти файлы не участвуют в текущей работе, а нужны как образец.
- `Работа с кастдевами/Транскрибации` — сюда кладём исходные расшифровки созвонов.
- `Работа с кастдевами/Сайт` — финальные красивые HTML-файлы сайта: `index.html` и страницы кастдевов.
- `Работа с кастдевами/Сайт/Служебные` — всё техническое для сайта: `json`, `js`, `css`, инструкции.

Правило именования:

- Транскрибация: `03_Транскрибация_Имя_Клиент.txt`
- HTML для внутренней работы: делаем страницу кастдева с тем же номером в логике задачи.

Важно:

- Номер у транскрибации и кастдева должен быть одинаковый.
- Следующий новый файл после `01` и `02` — это `03`.
- Сначала кладём транскрибацию в `Транскрибации`, потом на её основе делаем HTML с тем же номером.
- В папке `Сайт` держим только понятные HTML-файлы; всё служебное лежит внутри `Сайт/Служебные`.
